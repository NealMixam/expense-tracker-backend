# ⚙️ Expense Tracker — Backend

REST API для fullstack-приложения учета личных финансов.

---

## 🚀 О проекте

Серверная часть приложения, реализующая бизнес-логику, аутентификацию и работу с базой данных.

---

## 🛠 Стек технологий

- Node.js
- Express.js
- PostgreSQL
- pg — работа с БД
- jsonwebtoken — JWT-аутентификация
- bcryptjs — хеширование паролей

---

## 🧱 Архитектура

- REST API
- Middleware для авторизации
- Разделение слоев:
  - routes
  - controllers
  - services
  - database

---

## 🔐 Аутентификация

- JWT-токены
- Защита приватных маршрутов через middleware
- Хеширование паролей через bcrypt

---

## 📡 API возможности

- Регистрация пользователя
- Авторизация
- CRUD операций
- Работа с категориями
- Получение данных пользователя

---

## 🗄 Работа с базой данных

- PostgreSQL
- Проектирование структуры таблиц
- CRUD-запросы
- Связи между сущностями

---
